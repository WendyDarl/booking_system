# Secure Booking System

This project is a Secure Booking System developed using Django for the
Secure Software Development (SSD) assignment.

## Features
- User registration and login
- Booking creation and management
- User dashboard
- Audit logging

## Security Implementation
- Password hashing (Argon2)
- Login attempt limiting
- CSRF protection
- Secure session management

## Security Testing
- Static analysis using Bandit
- Dynamic testing using OWASP ZAP

## Tools Used
- Django
- SQLite
- Bandit
- OWASP ZAP
